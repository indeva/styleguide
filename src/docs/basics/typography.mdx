# Typography

In the Indeva Styleguide, the typography standardized using mixins on the default styles (`src/lib/styles/typography.scss`).

## Versions

We have 7 versions of typography, each one with bold version:

| Version     | Font Size | `@include mixing-name` |
|-------------|-----------|------------------------|
| Display 1   | 48px      | `@include display-1`   |
| Display 2   | 36px      | `@include display-2`   |
| Heading 1   | 24px      | `@include heading-1`   |
| Heading 2   | 20px      | `@include heading-2`   |
| Body        | 16px      | `@include body`        |
| Small Text  | 14px      | `@include small-text`  |
| Mini text   | 12px      | `@include mini-text`   |

To use the bold version using mixin, just pass the bold value as first argument on the mixin.

Ex: `@import display-1(bold)`
