# Icons

Here at Indeva we use icons through [FontAwesome](https://fontawesome.com/).

To use the icons, we use the official [Font Awesome React component](https://www.npmjs.com/package/@fortawesome/react-fontawesome).

## Usage

You can specify any icon from the [icons list](https://origin.fontawesome.com/icons):

<Playground className="separate-icons">
  <Icon icon="smile-beam" size="4x"/>
  <Icon icon="smile" size="4x"/>
  <Icon icon="laugh-squint" size="4x"/>
  <Icon icon="grin-stars" size="4x"/>
  <Icon icon="grin-squint-tears" size="4x"/>
</Playground>

## Prefixes

There's one another piece of magic that's happening in the background when providing icon names as strings like this: the `fas` prefix (for Font Awesome Solid) is being inferred as the default. Later, we'll look at what that means and how we can do something different than the default.

You have another prefixes to use, see it:

| Style                                               | Availability  | Prefix  |
|-----------------------------------------------------|---------------|---------|
| [Solid](https://fontawesome.com/icons?s=solid)      | Free          | `fas`   |
| [Regular](https://fontawesome.com/icons?s=regular)  | Pro Required  | `far`   |
| [Light](https://fontawesome.com/icons?s=light)      | Pro Required  | `fal`   |

### Examples:
<Playground className="separate-icons">
  <Icon icon={fasIgloo} size="3x" />
  <Icon icon={farIgloo} size="3x" />
  <Icon icon={falIgloo} size="3x" />
</Playground>

### Basic

[Size](https://fontawesome.com/how-to-use/on-the-web/styling/sizing-icons):

```javascript
<Icon icon="spinner" size="xs" />
<Icon icon="spinner" size="lg" />
<Icon icon="spinner" size="6x" />
```

[Fixed width](https://fontawesome.com/how-to-use/on-the-web/styling/fixed-width-icons):

```javascript
<Icon icon="spinner" fixedWidth />
```

Inverse:

```javascript
<Icon icon="spinner" inverse />
```

[List Items](https://fontawesome.com/how-to-use/on-the-web/styling/icons-in-a-list):

```javascript
<Icon icon="spinner" listItem />
```

[Rotate](https://fontawesome.com/how-to-use/on-the-web/styling/rotating-icons):

```javascript
<Icon icon="spinner" rotation={90} />
<Icon icon="spinner" rotation={180} />
<Icon icon="spinner" rotation={270} />
```

Flip horizontally, vertically, or both:

```javascript
<Icon icon="spinner" flip="horizontal" />
<Icon icon="spinner" flip="vertical" />
<Icon icon="spinner" flip="both" />
```

Spin and pulse [animation](https://fontawesome.com/how-to-use/on-the-web/styling/animating-icons):

```javascript
<Icon icon="spinner" spin />
<Icon icon="spinner" pulse />
```

[Border](https://fontawesome.com/how-to-use/on-the-web/styling/bordered-pulled-icons):

```javascript
<Icon icon="spinner" border />
```

[Pull left or right](https://fontawesome.com/how-to-use/on-the-web/styling/bordered-pulled-icons):

```javascript
<Icon icon="spinner" pull="left" />
<Icon icon="spinner" pull="right" />
```

Your own class names:

```javascript
<Icon icon="spinner" className="highlight" />
```
